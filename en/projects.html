Sugar Rocket. With Arduino.

Sugar rocket made at the start of 2021. The parachute ejection system is made with a servo that is activated by the Arduino.

Web server. With recycled parts.

Linux based web server running Docker on Fedora 38. Used to host this website as well as other services such as jellyfin or nginx. Video

Robot. 3D printed.

3D printed robot controlled by an arduino and moved by 8 servo motors. Use sensor in order not to crash into walls. Video

Fi-156 "Storch". RC airplane.

6-channel airplane built with foamboard and hot glue. It uses a 1000Kv motor, MG90S servos and it weights around 800 grams. Video

Satellite Images. With an RTL-SDR.

Satellite image received from Meteor M2-4 LRPT. This satellites transmit at 137.900MHz, I received the signal with a homemade QFH antenna and an RTL-SDR Blog V4. Video

INTEL 4004 emulator. Written in Rust.

INTEL 4004 emulator written in Rust. It's the first commercial microprocessor, it's a 4-bits CPU from 1971.
